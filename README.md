# 🧩 leetcode-challenges

Personal collection of LeetCode solutions organized by challenge. Each problem lives in its own subdirectory and every solution file is **standalone** — just run it directly, no external dependencies needed.

## 📁 Structure

```
leetcode-challenges/
├── 0001-two-sum/
│   ├── solution.js
│   ├── solution.py
│   └── solution.java
├── 0002-add-two-numbers/
│   └── solution.py
└── ...
```

## 🚀 Running a solution

Each file has its own `main` entry point and can be executed independently.

**JavaScript**
```bash
node 0001-two-sum/solution.js
```

**Python**
```bash
python 0001-two-sum/solution.py
```

**Java**
```bash
cd 0001-two-sum
javac solution.java
java solution
```
